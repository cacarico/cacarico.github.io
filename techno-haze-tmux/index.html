Techno Haze Tmux

Tmux has been one of the main tools I use daily and unfortunately configuring it to be on a minimum usable state takes some effort. After having a huge config file for years and suffering to manage it I decided to move it to it’s own plugin.

You can have it up and running by adding a single line in your tmux.conf.

set -g @plugin 'cacarico/techno-haze-tmux'

You can find the source code at https://github.com/cacarico/techno-haze-tmux

Features currently provided

  • Minimalistic Design: Clean UI with cyberpunk vibes.
  • Customizable Colors and Binds: Provides a better way to change UI and binds. (Not realeased yet)
  • Dynamic Status Indicators: Real-time updates on system resources, session info, and more.
  • Improved Default Keys: Iproved keybinds for navigation and management of tmux sessions, windows and panes.
  • Added functionalities: Added base functionalities to open projects, swap windows and more.
  • Lightweight: All the addons are designed to run light and fast.
