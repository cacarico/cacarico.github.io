Introducing Techno-Haze: A Tmux Plugin for Terminal Aesthetes

Hello, fellow terminal enthusiasts! ðŸ‘‹

If you’re anything like me, you spend more time customizing your terminal than you’d like to admit. Whether it’s tweaking NeoVim configurations or giving your Arch Linux setup that extra flair, the quest for the perfect terminal never ends. Today, I’m excited to share my latest creation with you: Techno-Haze, a tmux plugin designed to make your terminal sessions not just productive but also visually delightful.

Why Techno-Haze?

After over a decade in IT, migrating more than 30 companies to the cloud and wrestling with the intricacies of Kubernetes and Istio (our love-hate relationship is still going strong), I realized that the tools we use daily should bring us joy, not just efficiency. While tmux is an indispensable tool for multiplexing terminals, I felt it lacked a certain… pizzazz.

That’s where Techno-Haze comes in. Inspired by my love for fancy-looking terminals and a good techno beat, this plugin adds a sleek, modern aesthetic to your tmux status bar without compromising performance.

Features at a Glance

  • Minimalistic Design: A clean and unobtrusive look that enhances your terminal without overwhelming it.
  • Customizable Color Schemes: Match the plugin’s colors with your current theme or mood.
  • Dynamic Status Indicators: Real-time updates on system resources, session info, and more.
  • Lightweight: Written with efficiency in mind to keep your terminal snappy.

Installation Guide

Getting Techno-Haze up and running is a breeze. Here’s how:

Using Tmux Plugin Manager (TPM)

  1. Add the Plugin to Your .tmux.conf:
set -g @plugin 'cacarico/techno-haze-tmux'
  1. Press prefix + I (capital I) to fetch the plugin and source it.

Manual Installation

  1. Clone the Repository:
git clone https://github.com/cacarico/techno-haze-tmux ~/.tmux/plugins/techno-haze-tmux
  1. Source the Plugin in .tmux.conf:
run-shell ~/.tmux/plugins/techno-haze-tmux/techno-haze.tmux
  1. Reload Tmux Environment:
tmux source-file ~/.

Customization Options

Make Techno-Haze truly yours by tweaking its settings:

  • Change Color Scheme:
set -g @techno_haze_color 'purple'
  • Toggle Status Indicators:
set -g @techno_haze_show_cpu_usage 'on'
set -g @techno_haze_show_ram_usage 'on'
  • Adjust Refresh Interval:

set -g @techno_haze_refresh_rate '5'  # in seconds

[!NOTE] After updating your .tmux.conf, don’t forget to reload tmux…

tmux source-file ~/.tmux.conf
